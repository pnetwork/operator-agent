## 說明

此規格使用 JSON Schema 格式定義，主要包含以下必填的頂層屬性：

1. marketplace
- 市集顯示資訊
- 包含名稱、描述、版本等基本資訊
- 所有子屬性皆為必填

2. source
- 程式碼來源設定
- 定義程式碼儲存庫位置、認證方式和建置設定
- repository、build、registry 皆為必填

3. runtime
- 執行環境設定
- 指定程式語言、基礎映像檔和啟動命令等
- dependencies 為選填

4. installConfig
- 安裝配置
- required 為必填
- advanced 為選填
